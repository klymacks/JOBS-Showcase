# Push JOBS Showcase to GitHub

After creating the repository on GitHub, run these commands:

```powershell
cd C:\Users\rmason\Code\Repositories\KCAsrCB\JOBS-Showcase
git remote add origin https://github.com/klymacks/JOBS-Showcase.git
git branch -M main
git push -u origin main
```

Or if you prefer SSH:
```powershell
git remote add origin git@github.com:klymacks/JOBS-Showcase.git
git branch -M main
git push -u origin main
```
